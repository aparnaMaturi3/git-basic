Git Commands

Basic Commands

To initialize a project as a git repo - git init

To add a file to staging area - git add {file_name}

To add all the files to staging area - git add .

To add changes to your local repo - git commit -m "{message}"

To push committed files to local repo - git push

To ignore files/folder

Create a .gitignore file in the project folder and give the path of the directory or file that should be ignored

To revert all uncommitted changes only to specific file or directory - git checkout [somedir|file.txt]

To unstage all files that are staged with gid add - git reset and then git checkout file_name

To rename git command git config --global alias.my-add add
